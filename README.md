## 💻프로젝트 소개 
대학교에서 사용하는 학사정보시스템을 CRUD 중심으로 구현한 프로젝트입니다
  
## 🕰개발 기간
2024년 5월 2일 ~ 5월 19일 (총 18일)  
  
## ⚙개발환경
- Intellij IDEA Community
- Java 17
- Gradle 8.7
- Spring Boot 3.2.5  
  
## 📄기술 세부 스택
Spring Boot
- Spring Web
- Spring Data JPA
- Thymeleaf
- Spring Security
- PostgreSQL Driver
- Lombok
- Spring Boot Dev Tools
- Validation
- Java Mail Sender

## 📌주요 기능
- 학사 관리
- 수강 신청
- 성적 입력

## 📖공통 기능
로그인
- 세션 처리
- 아이디 찾기
- 비밀번호 찾기  
  
개인정보
- 개인정보 조회
- 개인정보 변경
- 비밀번호 변경
  
공지사항
- 공지사항 조회

## 👨‍🔧관리자 기능
사용자 관리
- 학생, 교수, 관리자 계정 CRUD
  
학사 관리
- 강의 CRUD
- 성적 CRUD
- 공지사항 CRUD
